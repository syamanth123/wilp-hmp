# HMP — Role Responsibilities

| Role | Responsibilities (per RFP) |
|------|---------------------------|
| **ADMIN** | Manage users, roles, permissions. Configure academic structure, workflows, notifications. Monitor audit logs and usage analytics. |
| **INSTRUCTION_CELL** | Initiate handout requests. Attach prior handouts. Fetch ERP data. Publish finalized handouts to LMS (Taxila). Manage exam/EC1 deadlines. |
| **HOG** | Review requests. Allocate faculty (off-campus/adjunct ≤ 3 courses/sem). Assign to Programme Committee. Provide final approval. |
| **PROGRAMME_COMMITTEE** | Receive HOG-allocated requests. Confirm faculty assignment. Run review/rework loop with faculty. Recommend or approve. |
| **FACULTY** | Apply for courses. View assignments. Edit Part B + evaluative components in the editor. Submit. Address rework comments. |
| **SME** | (Phase 2+) Nominated by Programme Committee for subject-matter changes. |
